Langkah Instalasi

Command Prompt (Admin)
REM 1. Jalankan installer sebagai Administrator
patriot_setup.exe

REM 2. Verifikasi instalasi
sc query PatriotUltraBlocker
Installer memverifikasi SHA-256 setiap binary sebelum ekstraksi. Jangan modifikasi atau repack file installer.

Geo IP + Time Control

Fitur baru di v1.0.1. Memungkinkan Anda memblokir koneksi masuk berdasarkan negara asal IP dan jadwal waktu (hari/jam). Sangat berguna untuk membatasi akses SMB/RDP dari luar Indonesia pada akhir pekan.

GeoIP lookup menggunakan ip-api.com (gratis, 45 req/menit). Hasil di-cache selama 1 jam. IP lokal (192.168.x, 10.x, 172.16-31.x) diizinkan otomatis tanpa lookup.

Referensi Hari

KodeHariContoh Penggunaan
0Minggurule0_days=0,6 (akhir pekan)
1Seninrule1_days=1,2,3,4,5 (hari kerja)
2Selasa
3Rabu
4Kamis
5Jumat
6Sabturule0_days=0,6 (akhir pekan)

Contoh Konfigurasi

config.ini — [geotime]
[geotime]
enabled=1
allow_countries=ID,SG,MY   ; ISO kode negara diizinkan
rule_count=3

; Rule 0: Blokir SMB dari luar negeri saat weekend
rule0_name=Block-SMB-Weekend
rule0_days=0,6          ; 0=Minggu, 6=Sabtu
rule0_h_start=0
rule0_h_end=23
rule0_ports=445,139
rule0_action=BLOCK

; Rule 1: Blokir RDP dari luar negeri saat weekend
rule1_ports=3389
rule1_action=BLOCK

; Rule 2: Alert RDP malam hari kerja (20:00-06:00)
rule2_name=Alert-RDP-Night
rule2_days=1,2,3,4,5
rule2_h_start=20
rule2_h_end=6
rule2_ports=3389
rule2_action=ALERT

Integrasi SIEM (Wazuh)

Koneksi plaintext ke remote host diblokir secara default. Selalu gunakan TLS via stunnel untuk mengirim event ke server Wazuh remote.

Konfigurasi Stunnel

stunnel.conf
[wazuh-tls]
client  = yes
accept  = 127.0.0.1:51514
connect = WAZUH_SERVER_IP:1514
CAfile  = /etc/stunnel/wazuh-ca.crt
verify  = 2

Config Patriot

config.ini
[siem]
siem_active=true
siem_host=WAZUH_SERVER_IP
siem_port=1514
siem_use_tls_proxy=true
siem_tls_local_port=51514

Cloud Update

Database IOC diperbarui dari cloud setiap 6 jam secara async. URL cloud tidak hardcoded — disimpan di file urls.dat terenkripsi DPAPI.

DatabaseFile LokalKonten
Applicationsapplication_security.csvHash + nama + kategori tool berbahaya
Ransomwareransomware_hash.csvHash + family + level severity
Network Denylistnetwork_denylist.csvIP/domain + kategori ancaman
Setiap file diverifikasi SHA-256 via sidecar file sebelum digunakan (fail-closed). Jika sidecar tidak tersedia atau hash tidak cocok, file lama dipertahankan.

Karantina

File berbahaya dikarantina dalam format .qtn — terenkripsi AES-256 dengan kunci berbasis ComputerName + UserName + install date. File tersimpan di %ALLUSERSPROFILE%\Patriot\quarantine\.

Operasi restore dan whitelist+restore memerlukan verifikasi password administrator. Operasi delete permanen memerlukan konfirmasi eksplisit.

AutoRollback

Snapshot dokumen diambil setiap 2–30 menit (konfigurasi via backup_interval_min). Saat ransomware terdeteksi dan diblokir, dokumen yang dimodifikasi selama proses berjalan dikembalikan otomatis dari snapshot terakhir.

Kuota backup dikontrol via backup_quota_mb (default 2048 MB). Saat kuota penuh, snapshot lama dihapus otomatis (FIFO).
