Multi-layer Protection

Each threat is handled by a different detection layer — from IOC hash to behavioral AI scoring.

Real-Time Process Monitor

Scan the new process every 3–15 seconds with an adaptive rate based on CPU load and threat level. Centralized snapshots prevent bottlenecks.

SHA-256 + SHA-1 Dual Hash Adaptive AI

Anti-Ransomware Engine

Detection of mass-file operations, high entropy, rename burst, and slow-encrypt patterns. AutoRollback restore documents in seconds.

🌏 Geo IP + Time Control

Block SMB/RDP connections from outside Indonesia on Saturday/Sunday. Up to 20 rules per installation with 1 hour GeoIP cache.

SIEM Integration

Send events to Wazuh via TLS proxy (stunnel). Format JSON, Windows Event Log ID 1001–1005. No cleartext to remote host.

Network Denylist

URLhaus feed from Abuse.ch is updated automatically. Non-blocking batch firewall — does not freeze the scan engine when blocking IP.

Attack Chain Detection

Multi-stage correlation: download → execute → lateral movement → exfil. Detect EDR killers, credential dumps, and C2 DGA beacons.

Honeypot Detection + AutoRollback

Hidden honeypot files in strategic locations (max 20/folder). Access = instant alert. AutoRollback restore malware-modified documents from encrypted snapshot backups.

AES-256 Quarantine Snapshot Backup Max 20 files/folder
