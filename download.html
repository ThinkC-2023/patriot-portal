Changelog & Audit

Semua perubahan, perbaikan bug, dan fitur baru. Termasuk hasil audit keamanan lengkap.

v1.0.1 28 Maret 2026 LATEST Unduh v1.0.1
FITUR BARU
  • NEW patriot_geotime.au3 — Modul Geo IP + Time Control. Blokir koneksi SMB/RDP berdasarkan negara asal dan jadwal hari/jam. Hingga 20 rules, cache GeoIP 1 jam, persist ke config.ini.
  • NEW Honeypot file limit — Maksimal 20 file honeypot per folder (HONEY_MAX_PER_FOLDER), total 100 file (HONEY_MAX_TOTAL). Counter dihitung dari tracker internal aHoneyFiles[].
CRITICAL FIX
  • C-01 SIEM cleartext diblokir — Koneksi plaintext ke remote host kini diblokir otomatis. Hanya loopback (127.x) yang diizinkan tanpa TLS. Tambahkan stunnel untuk enkripsi ke server remote.
  • C-02 DB integrity fail-closed — _VerifyCSVIntegrity(): semua 4 kondisi error (sidecar gagal, format invalid, hash gagal, hash mismatch) kini return False. Sebelumnya return True (fail-open).
HIGH FIX
  • H-01 Cloud URL tidak lagi hardcoded — URL Dropbox dipindah ke urls.dat terenkripsi DPAPI. Source code hanya berisi placeholder kosong.
  • H-02 InetGet blocking dihapus — _UpdateDBFromCloud() dan _SilentSync() menggunakan _InetGetWithFallback() async. Adlib timer tidak lagi dibekukan selama download.
  • H-03 SHA256 sidecar async — Download sidecar menggunakan InetGet mode=1 (async) dengan polling timeout 15 detik + retry 10 detik.
  • H-05 Legacy HTTP endpoint dinonaktifkan — _SendPatriotEvent() tidak lagi melakukan InetRead ke HTTP plaintext. Hanya TCP integration via PSK yang aktif.
MEDIUM FIX
  • M-01 JSON injection dicegah — _JsonEscape() ditambahkan. Semua field event (proc, reason, host) di-escape sebelum masuk ke JSON string.
  • M-02 Restore karantina butuh password — Operasi restore, whitelist+restore, dan multi-restore sekarang memerlukan verifikasi password via _PasswordDialog() + _VerifyPassword().
  • M-03 CSV write atomic — _RemoveFromQuarantineCsv() menulis ke .tmp dulu, baru atomic rename ke file asli. Mencegah data loss jika crash di tengah write.
  • M-04 Array OOB pada whitelist kosong — Guard if UBound($a) < 2 ditambahkan di _LoadWhitelist(). Counter $idx menggantikan index $i-1 yang rentan slot kosong.
  • M-05 DB diverifikasi sebelum dipakai — _EnsureDB() memanggil _VerifyCSVIntegrity() setelah setiap download. DB yang tidak terverifikasi ditolak.
LOW FIX
  • L-01 Hash cache di-cap 5000 entri — _SaveHashCache() membatasi output ke 5000 baris. Engine cycle cleanup: RemoveAll jika oHashCache.Count > 5000.
  • L-02 URLhaus download async — _LoadURLhausFeed() mengganti InetGet mode=0 dengan _InetGetWithFallback() timeout 20 detik.
  • L-03 Honeypot file limit — _CreateHoneyFiles() enforce HONEY_MAX_PER_FOLDER=20 dan HONEY_MAX_TOTAL=100.
  • L-04 oSlowEncrypt/oRenameBurst time-based eviction — RemoveAll diganti eviction selektif berdasarkan nilai/timestamp. Tracking aktif tidak terbuang.
  • L-05 SIEM log rotasi 10MB — patriot_agent.log dirotasi ke .old saat melebihi 10MB. File .old lama dihapus sebelum rename.
  • L-06 Firewall blocking non-blocking — _FirewallBlockIP() menggunakan batch queue. _FlushFirewallQueue() di-Adlib setiap 5 detik.
BUGFIX
  • BF-01 _EnsureDB() "If has no matching EndIf" — If _InetGetWithFallback(...) Then tidak memiliki EndIf pasangan. Diperbaiki: nested If FileExists() di dalam blok If, ditambahkan FileDelete($tmpVer) cleanup.
  • BF-02 URLhaus CSV parse konsistensi — StringSplit pakai "," biasa (bukan '","'), URL di $parts[2] sesuai _UpdateURLhausFeed(). Sebelumnya $parts[0] dengan strip quote manual.
  • BF-03 _PasswordDialog() return type — Fungsi mengembalikan array $ret[2]. Semua pemanggil sekarang bandingkan $res[0] bukan string "OK".
OPEN ISSUE
  • H-04 Installer hash masih kosong — Konstanta SETUP_HASH_PATRIOT_EXE, SETUP_HASH_ULTRA_EXE, SETUP_HASH_UNINST_EXE masih string kosong. Supply chain attack via installer tamper dimungkinkan. Dijadwalkan diperbaiki di v1.0.2.
v1.0.0 Januari 2026 STABLE Unduh v1.0.0
RILIS AWAL

  • INIT Honeypot detection dengan file umpan tersembunyi di 9 lokasi strategis.
