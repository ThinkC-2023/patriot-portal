// FITUR LENGKAP

Kapabilitas Deteksi

Setiap layer perlindungan dirancang untuk menangkap ancaman yang lolos dari layer sebelumnya — dari hash IOC hingga behavioral AI scoring.

// LAYER 1 — IDENTITAS

Deteksi Hash & Proses

SHA-256 + SHA-1 Dual Hash

Database IOC mendukung kedua format. SHA-256 sebagai primary, SHA-1 sebagai fallback. Cache hasil hash mencegah re-komputasi file yang sama — cap 5.000 entri.

Hash Cache5000 entry cap

Adaptive Scan Rate

Interval scan otomatis menyesuaikan: 3 detik saat threat aktif, naik ke 15 detik saat CPU di atas 75% atau idle. Menghemat resource tanpa mengorbankan respons.

3–15 detikCPU-aware

EDR Killer Detection

Tool yang dirancang mematikan AV/EDR mendapat skor 80 langsung — diblokir tanpa akumulasi. Deteksi via hash SHA-256 maupun SHA-1.

Score 80 instantNo accumulation

Process Reputation Score

Setiap proses memiliki skor reputasi kumulatif yang diperbarui lintas sesi. Reputasi di atas 40 memicu blokir langsung tanpa evaluasi lebih lanjut.

// LAYER 2 — PERILAKU

Behavior Engine

Anti-Ransomware — Multi-Signal Detection

Patriot mendeteksi ransomware dari berbagai sudut sekaligus. Tidak satu pun teknik tunggal — semua sinyal digabungkan ke dalam skor kumulatif per PID.

Mass file operation High entropy write Rename burst Slow-encrypt pattern Ransom note scan

LOLBin Detection

Monitoring 23 Living-Off-The-Land binaries termasuk PowerShell, Certutil, MSHTA, Rundll32, Regsvr32. Deteksi penggunaan tidak wajar berdasarkan konteks parent-child.

23 binariesParent chain

Attack Chain Correlation

Korelasi multi-stage: download file mencurigakan, eksekusi, lateral movement, dan exfiltrasi. Setiap stage menaikkan skor — pattern lengkap langsung diblokir.

Fileless Detection

Deteksi eksekusi in-memory via PowerShell encoded command, WMI subscription, script engine abuse. Tidak perlu file di disk — pattern perilaku sudah cukup.

Credential Dump Detection

Deteksi upaya membaca lsass.exe, dump SAM database, dan tool Mimikatz-family. Alert dengan dedup per-PID mencegah spam notifikasi.

// LAYER 3 — JARINGAN

Network Protection

Network Denylist (Blacklist)

Blokir IP/domain berbahaya via Windows Firewall API. Sistem batch non-blocking: IP dikumpulkan dalam antrian, di-flush setiap 5 detik. Tidak membekukan scan engine.

Batch flush 5sNon-blocking

URLhaus Feed (Abuse.ch)

Sinkronisasi otomatis setiap 6 jam dengan feed publik URLhaus — domain dan URL malware aktif. Zero konfigurasi, tidak butuh API key. Diunduh async tanpa memblokir engine.

Zero-configAsync download

C2 Beacon + DGA Detection

Heuristik koneksi periodik ke IP/domain eksternal — pola beacon C2. Ditambah deteksi domain algoritmik (DGA) berdasarkan entropi dan pola karakter nama domain.

Geo IP + Time Control — Fitur Baru

Blokir koneksi masuk berdasarkan negara asal IP dan jadwal waktu. Default: blokir SMB/RDP dari luar Indonesia pada Sabtu dan Minggu. Hingga 20 rules per instalasi dengan cache GeoIP 1 jam untuk efisiensi lookup.

20 rules max GeoIP cache 1 jam SMB 445 · RDP 3389 IP lokal exempt

Lateral Movement Detection

Deteksi PsExec, WMI remote execution, RDP brute force, dan pola perpindahan lateral dalam jaringan. Dedup alert per-session mencegah notifikasi berlebihan.

// LAYER 4 — RESPONSE

Proteksi & Recovery

Suspend → Kill → Quarantine

Proses berbahaya di-suspend dulu (freeze I/O) sebelum di-kill — mencegah ransomware menulis lebih banyak selama jeda. File dikarantina dengan enkripsi AES-256.

NtSuspendProcessAES-256 .qtn

AutoRollback

Snapshot dokumen otomatis setiap 2–30 menit (konfigurasi). Saat ransomware terdeteksi, dokumen yang dimodifikasi dikembalikan dari backup. Kuota backup 500MB–5GB.

Snapshot backup2–30 menit interval

Honeypot Detection

File umpan tersembunyi (hidden+system) di 9 lokasi strategis termasuk Desktop, Documents, Downloads, dan OneDrive. Akses = alert instan. Maksimal 20 file per folder.

9 lokasiMax 20/folder

Self Defense

Patriot memproteksi diri dari upaya terminasi. Deteksi Process Hacker, x64dbg, OllyDbg, dan 19 tool lainnya yang dipakai attacker untuk menghentikan guardian process.

22 kill toolsAnti-debug

Quarantine Management

GUI manajemen karantina: restore (dengan verifikasi password), hapus permanen, atau whitelist+restore. Multi-select dengan konfirmasi untuk operasi batch.

Whitelist Pintar

Whitelist per-path, per-hash SHA-256, atau per-folder. Verifikasi integritas saat load — slot kosong dari baris CSV kosong tidak dibaca sebagai path valid.
