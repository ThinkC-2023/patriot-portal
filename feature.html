// FULL FEATURES

Detection Capabilities

Each protection layer is designed to catch threats that bypass the previous layer — from IOC hash detection to behavioral AI scoring.

// LAYER 1 — IDENTITY

Hash & Process Detection

SHA-256 + SHA-1 Dual Hash

IOC database supports both formats. SHA-256 as primary, SHA-1 as fallback. Hash result caching prevents re-computing identical files — capped at 5,000 entries.

Hash Cache 5000 entry cap

Adaptive Scan Rate

Scan intervals automatically adjust: 3 seconds during active threats, increasing to 15 seconds when CPU exceeds 75% or system is idle. Saves resources without sacrificing responsiveness.

3–15 sec CPU-aware

EDR Killer Detection

Tools designed to disable AV/EDR receive an immediate score of 80 — blocked instantly without score accumulation. Detection via SHA-256 or SHA-1 hash.

Score 80 instant No accumulation

Process Reputation Score

Each process maintains a cumulative reputation score across sessions. Reputation above 40 triggers immediate blocking without further evaluation.

// LAYER 2 — BEHAVIOR

Behavior Engine

Anti-Ransomware — Multi-Signal Detection

Patriot detects ransomware from multiple behavioral signals simultaneously. No single technique — all signals contribute to a cumulative PID score.

Mass file operation High entropy write Rename burst Slow-encrypt pattern Ransom note scan

LOLBin Detection

Monitoring 23 Living-Off-The-Land binaries including PowerShell, Certutil, MSHTA, Rundll32, and Regsvr32. Detects abnormal usage based on parent-child execution context.

23 binaries Parent chain

Attack Chain Correlation

Multi-stage correlation: suspicious downloads, execution, lateral movement, and exfiltration. Each stage increases the threat score — full patterns trigger immediate blocking.

Fileless Detection

Detects in-memory execution via PowerShell encoded commands, WMI subscriptions, and script engine abuse. No disk file required — behavior patterns are sufficient.

Credential Dump Detection

Detects attempts to read lsass.exe, dump SAM databases, and tools from the Mimikatz family. Alerts are deduplicated per PID to prevent notification spam.

// LAYER 3 — NETWORK

Network Protection

Network Denylist (Blacklist)

Blocks malicious IPs and domains using the Windows Firewall API. Uses a non-blocking batch system — IPs are queued and flushed every 5 seconds.

Batch flush 5s Non-blocking

URLhaus Feed (Abuse.ch)

Automatic synchronization every 6 hours with the public URLhaus feed — active malware domains and URLs. Zero configuration and no API key required.

Zero-config Async download

C2 Beacon + DGA Detection

Detects periodic outbound connections indicating C2 beaconing. Includes detection of algorithmically generated domains (DGA) using entropy and character pattern analysis.

Geo IP + Time Control — New Feature

Blocks inbound connections based on country origin and time schedule. Default rule blocks SMB/RDP access from outside Indonesia during weekends.

20 rules max GeoIP cache 1h SMB 445 · RDP 3389 Local IP exempt

Lateral Movement Detection

Detects PsExec, WMI remote execution, RDP brute force attempts, and suspicious lateral movement patterns across the network.

// LAYER 4 — RESPONSE

Protection & Recovery

Suspend → Kill → Quarantine

Malicious processes are suspended first (I/O freeze) before termination — preventing ransomware from writing additional files during the kill phase. Files are quarantined using AES-256 encryption.

NtSuspendProcess AES-256 .qtn

AutoRollback

Automatic document snapshots every 2–30 minutes (configurable). When ransomware is detected, modified documents are restored from backup.

Snapshot backup 2–30 min interval

Honeypot Detection

Hidden decoy files (hidden + system attributes) placed in 9 strategic locations including Desktop, Documents, Downloads, and OneDrive. Access triggers an instant alert.

9 locations Max 20/folder

Self Defense

Patriot protects itself from termination attempts. Detects Process Hacker, x64dbg, OllyDbg, and 19 additional attacker tools used to terminate security processes.

22 kill tools Anti-debug

Quarantine Management

Quarantine management GUI: restore (password protected), permanent deletion, or whitelist + restore. Supports multi-select operations for batch actions.

Smart Whitelist

Whitelist by path, SHA-256 hash, or folder. Integrity verification is performed during load — empty CSV rows are ignored.
