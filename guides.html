Installation

Prerequisites

Component Minimum Version Description
Windows 10 / 11 (64-bit) Server 2019 / 2022 are also supported
Privileges Administrator Required for WMI, Firewall, and Event Log access
.NET Framework 4.x Included in Windows 10+
Internet Optional Required for cloud database updates

Installation Steps

REM 1. Run installer as Administrator
patriot_setup.exe

REM 2. Verify installation
sc query PatriotUltraBlocker
Installer verifies SHA-256 for every binary before extraction. Do not modify or repackage the installer.

Configuration

Main configuration file: %ALLUSERSPROFILE%\Patriot\config.ini

Key Default Description
scan_interval 8000 Normal scan interval (ms)
behavior_suspicious 15 Alert score threshold
behavior_block 30 Automatic block score
kill_cooldown 15000 Cooldown between killing the same PID
ai_block_score 20 AI automatic block score
log_retain_days 30 Delete logs older than N days
backup_quota_mb 2048 Rollback backup quota
backup_interval_min 5 Snapshot interval (minutes)

Modules

All modules can be enabled or disabled from the GUI (Settings > Modules) or directly from config.ini.

Geo IP + Time Control

New feature in v1.0.1. Allows blocking inbound connections based on country of origin and time schedule.

GeoIP lookup uses ip-api.com. Results are cached for 1 hour. Local IP ranges are automatically allowed.

SIEM Integration (Wazuh)

Plaintext connections to remote hosts are blocked by default. Always use TLS via stunnel.

Cloud Update

IOC databases are updated from the cloud every 6 hours asynchronously. Cloud URLs are stored in an encrypted file using DPAPI.

Quarantine

Malicious files are stored in encrypted .qtn format using AES-256 encryption.

AutoRollback

Document snapshots are created periodically. If ransomware modifies files, Patriot automatically restores documents from the latest snapshot.
